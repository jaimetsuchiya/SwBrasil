Main menu

All section in the page are listed as one page elements with the section titles on the left (My Profile, My Blog, Resume, My Portfolio, Contact Info).

Editing the Main menu
U can easily edit the Main menu item by replacing or editing the text between the <h2 class="item-title"> element.

<h2 class="item-title"><span class="title">my Profile</span> <span class="icon-user"></span></h2>

To make all sections expandable on page loading, not closed - do the following steps (index.php)

1. Find in the entire file item-title toggle noprint closed and replace with item-title toggle noprint opened
2. Find in the entire file item-title toggle closed and replace with item-title toggle opened
3. Find in the entire file class="item-cont clearfix" and replace with class="item-cont clearfix" style="display: block;" 3. Find in the entire file class="item-cont" and replace with class="item-cont" style="display: block;"

D) JavaScript - top


The theme makes use of various JavaScripts and the jQuery framework to add dynamic functionality. Some of the scripts used include:

1. Fancybox
2. HTML5
3. jQuery Easing
4. MIXITUP - A CSS3 and JQuery Filter & Sort Plugin
5. SlidesJS
6. LESS - Leaner CSS
If you would like to edit the javascript/jQuery code all js files are in folder "/js/".
