Getting started

Choose the HTML file called index.html and open it in any TEXT / HTML editor like Dreamweaver, notepad ++ etc.

HTML Structure

<header> contains the logo and Countdown.

<div class="quick-links"> has the icons of Envelope and Map. By clicking these icons page would scroll to the Newsletter and Google Map section.

<div id="accordion"> contains the 3 sectiosn of About Us, Team and Services section.

<footer> has 2 sections of Newsletter and Social Media Icons.

<div class="google-map"> has the Map obviously.

<div class="copyright"> contains the copyright info of your company.

Background Image

Backstretch is a simple jQuery plugin that allows you to add a dynamically-resized, slideshow-capable background image to any page or element. The image will stretch to fit the page/element, and will automatically resize as the window/element size changes.

In the index.html line no: 200 you can see the image path. Please set the new path of your images here.

Logo and Graphics

The complete set of 369 icons in Font Awesome 4.0.3 is used for all icons as well as Logo. You may change the text of logo and icon choosing from their iconset.

Time Countdown

A simple, amazing jQuery countdown timer plugin, visualized with jQuery Knob. Dates can be supplied in "DD-MON-YEAR-TIME" format.

In the index.html line no: 218 you can see $(".counters").ccountdown(2016,03,20,'12:00');. You need to set site launching time as well.

Accordion Section

This area is allocated for your company details i.e. About Us, Team and Services. You have to replace the Lorem Ipsum text with proper details in <p> tag and title in <h3> tag.

Social Links

The Subscribe form requires PHP support. You will need php mail function to get mails for user. The Icons are also Font Awesome Icons. You may add or delete any of these.

Google Map

Adding a Google Map to your web page is very easy. This web page contains a sample map embedded using the default embed code provided by Google Maps. The embedded map is non-responsive. So you have to set the iframe width="100%" and height="400px". You may increase the height.
